Primero istalase UFW

Comandos usados:

sudo apt-get update

sudo apt-get upgrade -y

which ufw

sudo apt-get install ufw

Explicación:

Os 2 primeiros comandos son para asegurar que a máquina está ben actualizada, "which" usouse para comprobar si ufw está inatalado e "sudo apt-get install"usouse para instalar ufw

Negar comunicación co exterior

Comandos usados:

sudo ufw default deny incoming

sudo ufw default deny outgoing

sudo ufw status verbose

sudo ufw enable

Explicación

Con "sudo ufw default deny incoming e outgoing" negamos toda conexión co exterior, con status verbose podemos miarar o estado do ufw e enable activa ufw, logo podemos comprobalo usando o comando ping

Logo usamos "sudo ufw reset" para devolvela a configuración orixinal

Averigua os portos abertos do teu router

Comandos usados:

sudo nmap localhost

Explicación

Co primer comando instalamos namp e co segundo podemos mirar os portos no noso router
